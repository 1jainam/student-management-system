# Student Management System

Full stack Student Management System built with React + Laravel.

## Tech Stack
- Backend: Laravel, JWT Authentication
- Frontend: React
- Database: MySQL

## Features
- Student CRUD (Create, Read, Update, Delete)
- JWT-based authentication
- RESTful API

## Setup Instructions

### Backend
1. `composer install`
2. Copy `.env.example` to `.env` and configure database
3. `php artisan migrate`
4. `php artisan serve`

### Frontend
1. `npm install`
2. `npm start`
